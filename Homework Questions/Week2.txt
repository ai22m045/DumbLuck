Describe a logging system with your own words:
A file/window/system that displays a variaty of messages (info/warning/error). These messages usually include the origin and the severity.


What is the difference between git submodules and git subtrees?
Git submodules are a link to a specific version another repository.
Git subtrees are a linkt to an entire other repository (including all the history).
